# Project Summary

HealthGuard is a Flask application that stores all runtime data in MongoDB.

## Core Features

- Password-based registration and login
- JWT-protected API routes
- Disease prediction with persisted history
- Health record storage
- ABHA token and health-data persistence

## Persistence Model

MongoDB collections:

- `users`
- `prediction_history`
- `health_records`
- `abha_tokens`
